<pre>

A primeira tag é a <pre>. Que resumidamente, é usada
para conter dentro dela um texto preformatado, ou
seja, mantém-se no formato escrito na tag

<img>

A Segunda tag apresentada é a tag <img>. Que
resumidamente, é usada para conter nela uma
imagem ou link que leve a uma imagem

<video>

A terceira tag apresentada é a tag <video>. Que
resumidamente, é usada para incorporar
conteúdo de vídeo por meio de links
