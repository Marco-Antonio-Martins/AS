<pre>

A tag <pre> define um texto como pré-formatado, ou seja,
mantém-se o formato escrito na tag, geralmente usado
para preservar os espaços e quebras de linha.

<img>

A tag <img> é usada para adicionar uma imagem
hospedada localmente ou um link que leve à
uma imagem alocada fora de sua página.

<video>

A tag <video> é quase igual à tag <img>, possuindo poucas diferenças,
quais sejam: ao invés de usar para imagem, usa-se para vídeos
e possui um atributo chamado controls, usado para adicionar
controles de reprodução ao vídeo.
