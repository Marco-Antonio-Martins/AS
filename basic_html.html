ESTRUTURA BÁSICA DE UM DOCUMENTO HTML5

ESTRUTURA BASE

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>

<!DOCTYPE html>

O <!DOCTYPE html> indica para o navegador qual a especificação de código utilizar. Que neste caso é o html

<html lang="pt-br">

A tag <html lang="pt-br"> indica para o navegador qual a linguagem utilizada no documento. Que neste caso é "pt-br"

<head>

A tag <head> contém informações legíveis apenas para a máquina (metadados) sobre o documento, como seu título, scripts e folhas de estilo.

PRINCIPAIS COMPONENTES DA TAG <head>

<meta charset="UTF-8">

Essa tag configura o conjunto de caracteres para "UTF-8". Isso significa que a partir de agora, o site reconhece o uso de acentuação.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Essa tag, resumidamente, indica para o navegador usar 100% da area de tela disponivel.

<title>Document</title>

Essa tag é usada para colocar o título do site. Que neste caso é Document.

<body>

A tag <body> significa o corpo do site, ou seja, tudo que aparecerá no site deve estar contida nessa tag.
