FORMATAÇÕES BÁSICAS DE TEXTO EM HTML5

Negrito

Para negritar uma frase, usamos a tag <strong>. Exemplo:

<p> <strong> Frase que deseja que fique em negrito </strong> </p>

Itálico

Para deixar uma frase em itálico, usamos a tag <em>. Exemplo:

<p> <em> Frase que deseja que fique em itálico </em> </p>

Marcador de texto

Para marcar uma frase, usamos a tag <mark>. Exemplo:

<p> <mark> Frase que deseja marcar </mark> </p>

Texto deletado

Para deixar um texto deletado, usamos a tag <del>. Exemplo:

<p> <del> Frase que deseja deletar </del> </p>

Texto inserido

Para deixar um texto como se tivesse sido inserido, usamos a tag <ins>. Exemplo:
<p> <ins> Frase que deseja que fique como inserida </ins> </p>

Texto sobrescrito

Para deixar um texto sobrescrito, usamos a tag <sup>. Exemplo:

<p> <sup> Frase que deseja que fique sobrescrita </sup> </p>

Texto subscrito

Para deixar um texto subscrito, usamos a tag <sub>. Exemplo:

<p> <sub> Frase que deseja que fique subscrita </sub> </p>
