ESTRUTURA BÁSICA DE TEXTO EM HTML5

CABEÇALHO

O elemento cabeçalho pode ser representado em níveis de 1 a 6, quanto menor o nível, maior
sua importancia. Sendo assim, o <h1> é o mais importante, e o <h6> é o de menor importância.
A tag cabeçalho é usada para " dizer ao navegador " que o que está dentro dela é um cabeçalho
Obs.: Mesmo mudando o tamanho da fonte dependendo do nível do cabeçalho, é errado usar essa
tag para aumentar ou diminuir a fonte, para isso temos tags específicas ou, melhor ainda, o CSS.

PARÁGRAFO

O elemento paragrafo pode ser representado pela tag <p>
A tag parágrafo é usada para agrupar conteúdos relacionados de qualquer tipo,
como imagens e campos de um formulário.
Obs.: Para pular linha dentro de uma tag paragrafo, usamos a tag <br>.
